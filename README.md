# Angular Boilerplate

> Boilerplate for Angular CLI.
## Prerequisites
- Node.js
- NPM - Node package manager

## Setting UP Environment
- git clone https://github.com/GitHubExperts-LK/angular-boilerplate.git

## Install Dependancies
```
npm install
```
## Run App
```
npm start
```
